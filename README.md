# LIRI Bot

A Command Line Interface App

## What does LIRI bot do?

LIRI can help search for Spotify songs, Bands in Town and OMDB for movies

Available commands:
* concert-this
* spotify-this-song
* movie-this

## How to Use

1. Clone Repo
1. Run npm install
1. Command prompt: *node liri.js* '< use a command above >'


### Technologies
* NodeJS
* JavaScript
* Spotify API
* OMDB API
* Bands In Town API
* NPM: request, dotenv
